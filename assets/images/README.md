# Background photos

- forest-1.jpg, forest-2.jpg, forest-3.jpg
- sea-1.jpg, sea-2.jpg, sea-3.jpg
- fireplace-1.jpg, fireplace-2.jpg, fireplace-3.jpg
- tropical-beach-1.jpg, tropical-beach-2.jpg, tropical-beach-3.jpg
- canadian-forest-1.jpg, canadian-forest-2.jpg, canadian-forest-3.jpg

Calm, low-contrast landscapes work best (they sit under white UI text and the progress bar). Until a file is present, the app shows a soft gradient placeholder in its place — no code changes needed once you drop the real files in.
